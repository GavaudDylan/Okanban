## Falsy values

En JS, toutes les valeurs ont un équivalent `true` (truthy) / `false` (falsy).

Tout le monde est truthy sauf 6 valeurs : 
- `false`
- `null`
- `undefined`
- `""`
- `0`
- `NaN`

### Exemple

```js
false || true   // true

false || 1      // 1

undefined || 1  // 1
```
